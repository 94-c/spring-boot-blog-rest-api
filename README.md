# [토이] Spring Boot Jpa를 활용한 REST-API
- 사내 스터디를 통한 Spring Boot JPA를 활용 REST-API

## 프로젝트 세팅
### 📌 프로젝트 스펙
- Java 17
- SpringBoot 2.7.12
- Dependencies 
  - Spring Data Jpa
  - Lombok
  - Spring Web
  - MySQL Driver
  - Validation
  - Srping Boot Security
  - Jwt
    - jjwt-api
    - jjwt-impl
    - jjwt-jackson
- Spring Boot Mail
